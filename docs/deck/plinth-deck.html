The trust gap blocks AI trading from scaling

Agents on Arc run on their own balance. Taking outside capital is blocked by two failures.

1. Custody
Investors must hand keys or signing authority to an unknown agent.

Prior protocols (Enzyme, dHEDGE) custody via multisig — heavy, slow, requires KYC infrastructure.

Plinth: capability-not-custody — agent directs funds, can never withdraw to new addresses.
2. Verification
When an agent reports profit, the only "proof" is the agent's own message on chain.

Off-chain venues (CEXes, Aster, Hyperliquid) leave depositors trusting the agent's word.

Plinth: cryptographic Underwriter reconciles agent's claim against venue's on-chain trade history.

Both must be solved at the infra layer. Plinth is that layer.
